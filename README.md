# Project Overview

Mint Classics is evaluating whether one of its warehouses can be closed while maintaining healthy customer service levels.

This project analyzes inventory, sales, and warehouse data using MySQL to identify opportunities for warehouse consolidation/closure.

## Tools Used

- MySQL
- MySQL Workbench
- SQL
- Google Slides

## Analysis Performed

- Compared warehouse capacity utilization
- Calculated inventory value by warehouse code
- Analyzed sales volume and revenue
- Calculated inventory turnover
- Identified slow-moving products

## Key Findings

- Warehouse B generates the highest revenue and sales volume and should remain open.
- Warehouse C is the strongest candidate for closure due to its lower revenue contribution and the fact that it only uses half of its full storage capacity.
- Several slow-moving products may represent excess inventory.

## Recommendation

Closue Warehouse C after reviewing inventory relocation plans and reducing excess stock.
